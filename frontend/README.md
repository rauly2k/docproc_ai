# Document AI Frontend

React + TypeScript + Vite frontend for the Document AI platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- Chat with PDF using RAG (Retrieval-Augmented Generation)
- Document selection for targeted search
- Real-time answers with source citations
- Model quality selection (Flash/Pro)
